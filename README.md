## Gains
- Tailwind development
- React functionality
- Next.js project development
- Vercel deployment via GitHub
- ESLint assessment

## Usage/How to run
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
